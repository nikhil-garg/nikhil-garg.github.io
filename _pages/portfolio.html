Selected research and engineering projects across neuromorphic hardware, brain-computer interfaces, embedded systems, and bio-signal processing.

UNICO

Unsupervised spiking neural networks with analog memristive devices for edge computing. This project developed voltage-dependent synaptic plasticity rules, memristive synapse models, simulation workflows, and mixed-signal ASIC prototypes for local learning in spiking neural networks.

  • Project page
  • VDSP_Nengo: VDSP implementation in the Nengo SNN simulator.
  • VDSP-Memristors: code for VDSP with resistive and ferroelectric synapses.

IONOS

Iono-electronic neuromorphic interfaces for communication with living systems. Work included analog-to-spike conversion, reservoir-computing workflows for EMG hand gesture recognition, and evaluation frameworks for organic electrochemical transistor reservoirs.

  • Project page
  • EMG_exp: code for signals-to-spikes EMG reservoir computing.

Brain-Computer Interfaces

Real-time EEG and machine-learning pipelines for assistive technology, motor imagery, biometric authentication, cognitive-state estimation, and portable-headset emotion decoding.

  • Decoding_EEG: code for decoding valence and arousal from portable EEG.
  • Hybrid SSVEP and eye-blink wheelchair control for home-environment navigation.
  • Motor-imagery EEG pipelines for multiclass BCI and biometric authentication.

FPGA and Embedded Hardware

FPGA, PCB, and hardware-software co-design work for memory characterization, embedded acceleration, and signal-processing systems.

  • FPGA-controlled characterization PCB for non-volatile memory devices.
  • SPI/I2C hardware interfaces for ADC, DAC, OLED, and FPGA-board integration.
  • Reconfigurable hardware accelerator work for embedded FPGA systems.

VITFOX

Ferroelectric-oxide hardware and algorithm work for neural-network training, including energy-convergence tradeoffs and bio-inspired learning strategies on emerging devices.

  • EEG-Ferro: ferroelectric-synapse EEG processing work.
  • Energy-convergence-Ferro-CCMCC2026: energy-convergence tradeoff for neural-network training on bio-inspired hardware.

SWIMS

Current postdoctoral work in the Bio-Inspired Circuits and Systems group at the University of Groningen, in the scope of the ERC project SWIMS: Stochastic Spiking Wireless Multimodal Sensory System.
