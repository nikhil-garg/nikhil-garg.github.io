Key projects
======
* **UNICO:** Developed unsupervised spiking neural network learning rules, hardware-compatible synaptic plasticity algorithms, system-level simulation frameworks, and mixed-signal ASIC prototypes for memristor-integrated neuromorphic systems.
* **IONOS:** Built analog-to-spike signal encoding workflows and reservoir-computing techniques for EMG processing, later extending evaluation frameworks with organic electrochemical transistor reservoir models.
* **Brain-computer interfaces:** Developed real-time EEG signal processing and machine learning pipelines for wheelchair automation, motor imagery, biometric authentication, emotion detection, and speech-based Alzheimer's detection.
